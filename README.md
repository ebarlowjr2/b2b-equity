# b2b-equity

Equity-for-support marketplace + cap table ledger.

## Stack
- Next.js (web)
- Supabase (auth, database, storage, edge functions)
- Expo (mobile)
- Vercel (web deploy)

## Monorepo Layout
- `apps/web` — Next.js (App Router)
- `apps/mobile` — Expo
- `packages/ui` — shared UI components (optional)
- `packages/lib` — shared types/helpers (optional)
